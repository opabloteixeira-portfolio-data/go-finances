# 💰💵 goFinances

### Aplicação desenvolvida em ReactJS para controlar entradas, saídas e balanço de uma aplicação web financeira

## 🗒️ Features do projeto 🗒️

- Entradas
- Saídas
- Total
- Importar CSV
- Site responsivo
